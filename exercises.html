Hello Friends, this Typing Practice System is specially developed for the PSSSB Typing Examination.

These mock tests are designed according to the latest Punjab SSSB typing guidelines.

  • To qualify, you must type at least 1500 characters with a minimum of 92% accuracy.
  • Please use Google Chrome browser for the best performance.
  • Press Enter or the Down Arrow key to move to the next chunk (only after typing 80% of the current chunk).
  • Use the Page Up and Page Down keys to move between chunks.
  • An option is available to disable the Backspace key during practice to help improve typing accuracy.
  • If you face any issue, please email us with a screenshot for support.

Practice daily and improve your accuracy before the final examination.
